# 🎓 Smart Learning Management System (LMS)

## 📌 Project Overview
The Smart Learning Management System (LMS) is a web-based platform designed to support online learning. It enables students to enroll in courses, access learning materials, submit assignments, and track their academic progress. Instructors can create and manage courses, while administrators oversee the entire system.
